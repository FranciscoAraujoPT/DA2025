Detailed Description

Provides functions to plan hybrid (car + walking) routes with an emphasis on environmental friendliness.

This namespace includes algorithms that find optimal paths from a starting point to a destination, prioritizing minimal environmental impact by promoting walking for the last portion of the trip.

Finds the best environmentally friendly route that combines driving and walking.

This function calculates the best route from a starting point to a destination using a hybrid approach: driving to a nearby parking spot and walking the rest of the way. It prioritizes minimizing total travel time while respecting a maximum walking time constraint.

Parameters
