Detailed Description

template<class T>
class Graph< T >

Represents a generic graph structure using adjacency lists.

Supports vertex and edge management, as well as support for bidirectional and weighted edges (walking and driving). It also includes auxiliary fields for pathfinding algorithms such as Floyd-Warshall.

Template Parameters
TThe type of data stored in each vertex.
