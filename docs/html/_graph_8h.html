Detailed Description

Contains the template definitions for the Graph, Vertex, and Edge classes.

This header provides a flexible graph data structure supporting directed and bidirectional edges, as well as weighted paths for multiple transportation modes (e.g., walking and driving). It includes all core components needed for graph-based algorithms and routing logic.
