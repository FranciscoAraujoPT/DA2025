Detailed Description

template<class T>
class MutablePriorityQueue< T >

A priority queue that allows updating the priority of its elements.

This class supports efficient key updates, which is useful for algorithms like Dijkstra.

Template Parameters
TThe type of elements stored in the queue. Must have an int queueIndex and support comparison with <.
