Computes the shortest path between two vertices using Dijkstra's algorithm.

This function returns the shortest path from a source vertex to a destination vertex in the given city graph. The graph is assumed to represent a weighted, directed graph with non-negative edge weights.

Parameters
cityPointer to the graph representing the city.
srcThe starting vertex.
destThe destination vertex.
currentBestOptional upper bound on path cost; paths with a cost exceeding this value are skipped, enabling early exit and optimization.
isWalkingIndicates whether walking time should be prioritized (true) or driving (false).
Returns
A vector of vertices representing the shortest path from src to dest.
