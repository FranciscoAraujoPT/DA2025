DA - Delivery Truck Pallet Packing Optimization
A delivery truck packing optimization system that maximizes profit while respecting weight constraints using classic and modern algorithms. This system reads pallet and truck data from CSV files and applies a variety of optimization techniques—including Brute Force, Dynamic Programming, Greedy Approximation, and Integer Linear Programming—to solve a knapsack-like problem. It provides an interactive menu interface for selecting datasets and algorithms, making it an educational and practical tool for analyzing different algorithmic strategies in combinatorial optimization.

Detailed Description

Implements the greedy approximation approach for solving the knapsack problem.

This file contains the implementation of the algorithm that uses a greedy approach based on the weight-to-profit ratio to select pallets. It provides a fast but approximate solution to the problem.
