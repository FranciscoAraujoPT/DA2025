Detailed Description

template<class T>
class Vertex< T >

Represents a vertex (node) in a graph.

Each vertex holds information of generic type T and maintains a list of outgoing and incoming edges. It also includes various auxiliary fields for graph algorithms like DFS, BFS, Dijkstra, and Tarjan's SCC.

TThe data type stored in the vertex.
