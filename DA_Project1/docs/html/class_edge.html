Detailed Description

template<class T>
class Edge< T >

< Required for priority-based graph algorithms (e.g., Dijkstra).

Represents an edge (connection) between two vertices in a graph.

Forward declaration of Edge class.

Needed because Vertex and Edge reference each other.

Template Parameters
TType stored in the graph nodes.

Each edge supports two weight types: one for walking and one for driving. Edges can also be marked as available or unavailable and support bidirectional links.

Template Parameters
TThe type of data stored in the vertices this edge connects.
